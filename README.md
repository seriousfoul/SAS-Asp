這是一個使用 Python爬蟲 進行取得網站資料後，
存入雲端 MySql，
用Asp.Net MVC 網站的方式，來呈現資料

- 首頁：
分為兩大類，一類是當日銷售額高的遊戲，另一類是玩遊戲的人數。
呈現每個月：
上榜次數前五名遊戲
上榜次數前十名遊戲類型
上榜次數前五名的遊戲售價類型

- 今日暢銷遊戲：
當日銷售額高的遊戲，前25名

- 今日遊玩人數排行榜：
當日玩遊戲的人數，前25名

- 遊戲相關統計表：
繪製遊戲類型文字雲
上榜前五名遊戲的當月排名 - 走勢圖
上榜次數前十名遊戲類型 - 長方圖
上榜次數前五名遊戲的售價類型 - 圓餅圖

- 串接資料：
將過往的所有排名資料，
以API的形式提供給需要資料的人。

- 登入與註冊：
使用者登入與註冊來使用討論區功能。

- 討論區：
需要登入帳號，才能發表與回覆文章。
未登入的使用者僅能看文章。

- 購物商城：
未註冊前可將商品加入於購物車中，
結清費用需要登入帳號。
